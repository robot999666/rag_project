# 附：高阶课程介绍

# 《中国文化》课程介绍

本课程旨在介绍中国文化的脉络、思潮及特征，使学生对于中国文化具有系统的、深层次的理解。课程内容包括中国文化的思想来源、中国文化的传播及发展、中国文化与西方文明的交汇、中国文化在当代社会的实践意义等。课程的主要目的在于使学生对于中国文化产生感性与理性兼具的审美体验，提高学生对于中西方文化异同的理解能力，培养学生将中国传统智慧与实践相结合的能力。

# 《欧洲文化入门》课程介绍

本课程的定位是一门内容广泛的入门课，通过学习本课程，希望学生对欧洲文化能有初步的了解，为他们更深入的学习和了解欧洲文化打下基础。

该课程主要内容包括欧洲文化的重要方面，如希腊、罗马文化、中世纪、文艺复兴与宗教改革、十七世纪、启蒙运动、浪漫主义、马克思主义与达尔文学说、现实主义、现代主义及其它。

# 《学术英语阅读》课程介绍

学术英语阅读(社科): 阅读经济学，社会学，媒体，政治学学科经典教材的部分章节，了解四门学科的一些基本理论和概念，练习从多学科视角分析社会现象，培养批判性思考，独立思考，创新思考能力。要求: 较强阅读与写作能力。课程要求:笔试，课上发言讨论，需完成一篇研究性论文。

# 《学术英语视听说》课程介绍

学术英语视听说课程重视学生听说能力的培养，聚焦全球化时代下学生英语应用能力的塑造，满足大学生英语听说能力发展的现实需求与未来需求，逐步发展学生的自主学习能力，增强跨文化交际意识和交际能力，提高综合文化素养和人文素质。

# 《学术英语翻译》课程介绍

《学术英语翻译》开课一学期，共计 32个学时，每周2个学时。

该课程旨在通过系统的学习培养学生对英汉语言文化差异的敏感性，初步了解英汉翻译的基础理论和英汉两种语言的异同，并掌握常用的英汉翻译技巧，提高学生的翻译能力。此外，该课程会对各种文体，尤其是学术类文章进行分析，使学生了解不同文体大致特征及其翻译时应注意的事项，从而在不同文体的翻译中能有的放矢。

# 《中西文化比较》课程介绍

《中西文化比较》是一门重点介绍两个主要的英语文化国家—英国和美国—和中国的基本文化知识的中西文化概况类课程。本课程的授课对象是已经完成大学英语基础阶段课程的学习，达到大学英语四级以上水平，需要进行后续拓展文化知识学习的学生。该课程旨在帮助学生扩大对中西文化基础知识的掌握，熟悉相关的英文表达，使他们能够用恰当的英文进行表述，提高对跨文化知识的实际运用能力，最终提高学生的跨文化交际能力。该课程采用国内比较权威，由南京大学出版社出版的全英文教材《中西文化概况英文版》。教学方法采用研讨式，学生课外阅读，微课，与课堂小组演示和讨论相结合。总评成绩采用形成性评估方式，平时成绩和期末考试成绩各占一定比例。

# 《高级交流技巧》课程介绍

《高级交流技巧》是为非英语专业学生开设的一门后续选修课，旨在通过国际化的资源和视野，多角度的思维方式和实用的学习方法，提升学生在国际场景与多元文化中的英语听说交流能力，形成看、听、想、说自然融为一体的英语语言交流模式，即，通过观看国外原声视频材料，了解英美世界的社会发展与历史文化，拓展学生的国际视野；通过泛听与精听练习相结合的方式，训练学生的听力技巧，提升学生的听力理解能力；通过复述、情景表演、讨论、辩论等目标任务，培养学生的批判性思维能力，提高其口语表达能力，最终完成英语视听输入与英语口语输出的交流实践目的。

# 《大学法语Ⅰ》课程介绍

大学法语Ⅰ是法语入门课程，使用的教材是孙辉编著的《简明法语教程》上册。教材共30个单元，包括 3部分，即语音教程（1-8单元）、入门教程（9-18单元）、和初级教程（19-30 单元）。本学期主要进行语音部分的教学（1-8 单元）,掌握法语中 35 个音素的发音规则，其中 15 个元音音素，17 个辅音音素，3个半元音或半辅音。由于英语和法语同属印欧语系，两种语言互相借鉴、互相

渗透，在语音上有许多相似之处，因此教学中主要采用英法对照法，充分利用语言学习中的正迁移现象，尽快使学生掌握法语的发音技巧、熟悉法语字母和音标的书写，并熟练使用 1-8单元中的句型进行交流。

# 《大学日语 I》课程介绍

课程目标：日语入门以及掌握一些基础的日语词汇、语法、表达与日常对话。

课程内容：入门单元包括日语发音、日语文字的书写、声调与语调等。基础日语单元包括常见单词、基础语法、惯用表达、日常对话等，同时也会适当补充一些日本的语言习惯、风土人情等方面的介绍。

课程教材：新版中日交流标准日本语初级（上）。

# 《大学俄语 I》课程介绍

《大学饿语 I》是为非英语专业学生开设的高阶选修课程，其目的是培养学生的俄语的基础综合应用能力，使他们在今后学习、工作和社会交往中能用俄语进行简单交际，提高其综合文化素养和跨文化交际能力，以适应我国社会发展和国际交流的需要。

本课程主要选择大学生活、学习以及一般性社会场景，帮助学生了解俄语基本语音语调，掌握基本语法知识，基本形成俄语自主学习能力，为日后的俄语学习奠定基础。

# 《实用商务英语》课程介绍

本课程为大学英语高阶课程中的特色课程。通过在不同商务话题下对英语听、说、读、写、译五方面的训练和相关话题篇章的讲解分析，使学生能够逐步提高商务语篇阅读理解能力，扩大商务英语专业词汇量和熟悉商务英语常用句型，具备基本的口头和笔头表达能力，培养和提高学生综合运用英语的能力。课程注重学生团队协作能力的培养，让学生分组完成商务写作、案例分析、商业调研等任务。
