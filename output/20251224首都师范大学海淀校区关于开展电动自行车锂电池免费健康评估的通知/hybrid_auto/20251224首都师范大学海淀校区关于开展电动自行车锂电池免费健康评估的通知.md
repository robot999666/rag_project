# 首都师范大学海淀校区

# 关于开展电动自行车锂电池免费健康评估的通知

海淀各校区全体师生员工：

为进一步筑牢校园电动自行车安全防线，有效防范因电池老化、故障等引发的安全事故，切实保障广大师生员工生命财产安全，根据《北京市非机动车管理条例》“电动自行车电池超过生产日期五年的，应当进行安全性评估；未通过安全性评估的，应当停止使用”相关规定，学校联合八里庄街道在校本部、北一区组织开展电动自行车锂电池免费健康评估活动，现将具体安排通知如下：

# 一、评估范围

海淀各校区师生员工的已经安装校制电子号牌的电动自行车锂电池，应检尽检。

# 二、评估时间及地点

# 三、评估内容

（一）电池外观检查（是否存在鼓包、破损、漏液等情况）；  
（二）电池电压、内阻评估（判断电池健康状态）；  
（三）充电接口安全性评估；

（四）电池续航能力评估及养护建议。

# 四、参与方式

本次活动为免费公益性质，无需提前报名。请广大师生员工按照评估时间自行骑行电动自行车至评估地点进行评估。

# 五、温馨提示

（一）请广大师生员工根据自己时间安排，合理选择评估地点，避免跑空；  
（二）评估过程中请遵守现场秩序，依次进行、避免拥挤，自觉维护环境卫生；  
（三）此次活动未参与评估的师生员工，学校将不再集中组织评估，请关注其他相关通知另行评估。  
（四）如遇问题，请拨打联系电话：010—68902751。
